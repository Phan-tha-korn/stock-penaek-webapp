# คู่มือการใช้งานระบบ Stock Penaek

เอกสารชุดนี้จัดทำไว้สำหรับอธิบายการใช้งานระบบแบบแยกตามบทบาทผู้ใช้ เพื่อให้เปิดอ่านแล้วเข้าใจได้ทันทีว่า:

- แต่ละ `User Role` เห็นเมนูอะไรบ้าง
- แต่ละเมนูใช้ทำอะไร
- ต้องกดตรงไหนก่อนหลัง
- งานแต่ละส่วนเชื่อมกันอย่างไร
- ระบบทั้งหมดย้อนกลับไปดู flow การทำงานได้อย่างไร

## ใช้เอกสารนี้เมื่อไร

- ตอนเริ่มใช้งานระบบครั้งแรก
- ตอนสอนงานพนักงานใหม่
- ตอนเช็กว่าแต่ละ role ควรทำอะไรได้หรือไม่ได้
- ตอนวางลำดับการตรวจสอบงานจริงก่อนใช้งาน production

## เมนูหลักที่มีในระบบตอนนี้

- ค้นหาสินค้าและเช็ก
- เทียบสินค้าแบบไว
- คิวตรวจสอบอัตโนมัติ
- การแจ้งเตือน
- จัดการสินค้า
- จัดการร้านค้า
- ธุรกรรม
- รายงาน
- ตั้งค่าและหน้าตา
- เครื่องมือพัฒนา
- จัดการผู้ใช้
- มุมมองเชิงลึก

## หลักการสำคัญของระบบ

- งานสำคัญต้องมีประวัติย้อนหลัง
- ข้อมูลที่ใช้อนุมัติย้อนหลังต้องตรวจสอบซ้ำได้
- การเปลี่ยนแปลงสำคัญต้องมี notification และ audit
- การค้นหาและเทียบราคาควรใช้ได้แม้พิมพ์ไม่ครบ
- งานรอตรวจสอบควรเด้งขึ้นมาให้อัตโนมัติ ไม่ต้องค้นหาก่อนเสมอ

## หมายเหตุ

- ชื่อเมนูในแต่ละ role อาจต่างกันเล็กน้อยเพื่อให้ตรงกับงานของ role นั้น
- บางหน้าเป็น read-only สำหรับบาง role แม้จะมองเห็นเมนูเดียวกัน
