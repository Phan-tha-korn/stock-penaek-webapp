# คู่มือการใช้งานสำหรับ STOCK

บทบาท `STOCK` เน้นใช้งานจริงหน้างานแบบอ่านข้อมูล ค้นหา เปรียบเทียบ และตรวจสอบสินค้าใน branch ที่ตนมีสิทธิ์

## 1. เมนูที่ STOCK เห็น

### เมนูหลัก
- `ค้นหาสินค้าและเช็ก`
- `เทียบสินค้าแบบไว`
- `การแจ้งเตือน`

### ข้อมูลงาน
- `จัดการสินค้า` หรือ `รายการสินค้า`
- `รายชื่อร้านค้า`
- `ธุรกรรม`

### การตั้งค่า
- `ตั้งค่าหน้าตา`

## 2. เมนูหลักแต่ละตัวใช้ทำอะไร

### 2.1 ค้นหาสินค้าและเช็ก
ใช้ค้นหาสินค้าแบบทั่วไปเพื่อดูข้อมูลเร็ว

ใช้เมื่อ:
- อยากเช็กว่ามีสินค้านี้ไหม
- อยากดูข้อมูลเบื้องต้นของสินค้า
- อยากเปิดจากชื่อสินค้า/รหัสสินค้าแบบเร็ว

### 2.2 เทียบสินค้าแบบไว
เป็นเมนูสำคัญมากของ STOCK

ใช้เพื่อ:
- พิมพ์คำคร่าว ๆ แล้วให้ระบบแนะนำสินค้า
- เลือกหลายสินค้าเพื่อเทียบพร้อมกัน
- ดูว่าตอนนี้สินค้าตัวไหนราคาดีสุด
- เปิดดูรายละเอียดสินค้าได้ทันที

วิธีใช้งานแบบง่าย:
1. เปิด `เทียบสินค้าแบบไว`
2. พิมพ์คำค้นหา เช่น
   - รหัสสินค้า
   - ชื่อสินค้า
   - alias
   - ชื่อร้านค้า
3. ระบบจะเด้งรายการให้เลือก
4. กดเลือกสินค้าอย่างน้อย 2 รายการ
5. ใส่จำนวนที่ต้องการใช้เทียบ
6. ระบบจะสรุปผลให้อัตโนมัติ
7. ถ้าต้องการดูลึก ให้กด `รายละเอียด`

ผลที่ควรดู:
- ร้านที่ดีที่สุดตอนนี้
- ต้นทุนรวมที่ใช้เทียบ
- ราคา base
- วิธีรับของ / พื้นที่

### 2.3 การแจ้งเตือน
STOCK เห็นเฉพาะสิ่งที่เกี่ยวกับงานของตนตามสิทธิ์

ใช้เพื่อ:
- ดูว่ามีเรื่องใหม่เข้ามาไหม
- เปิดไปยังหน้าที่เกี่ยวข้องอย่างรวดเร็ว

### 2.4 รายการสินค้า
ใช้ดูรายการสินค้าและเปิดรายละเอียดสินค้า

### 2.5 รายชื่อร้านค้า
ใช้ดูข้อมูลร้านค้าที่เกี่ยวข้องกับการทำงาน

### 2.6 ธุรกรรม
ใช้ดูข้อมูลการเคลื่อนไหวที่เกี่ยวกับงานสต็อก

### 2.7 ตั้งค่าหน้าตา
ใช้ปรับ:
- ภาษา
- โหมดสี
- การแสดงผลส่วนตัว

## 3. สิ่งที่ STOCK ควรทำเป็นประจำ

1. เปิด `ค้นหาสินค้าและเช็ก`
2. เปิด `เทียบสินค้าแบบไว` เมื่อจะตัดสินใจเทียบของ
3. ดู `การแจ้งเตือน`
4. เปิด `รายการสินค้า` เพื่อดูรายละเอียดหรือเช็กข้อมูลประกอบ

## 4. สิ่งที่ STOCK ทำไม่ได้

- อนุมัติ verification
- เปลี่ยน matching สำคัญ
- จัดการ flow ระดับระบบ
- ดูข้อมูลข้าม branch ถ้าไม่ได้รับสิทธิ์

## 5. ตัวอย่างการใช้จริง

### ต้องหาอะไหล่ที่ราคาดีและเลือกเร็ว
1. เปิด `เทียบสินค้าแบบไว`
2. พิมพ์คำค้นหา
3. เลือกสินค้าที่ใกล้เคียงกัน 2 รายการขึ้นไป
4. ดูสรุป
5. กดรายละเอียดตัวที่สนใจ

### ต้องเช็กว่าสินค้าที่ทีมพูดถึงคือตัวไหน
1. เปิด `ค้นหาสินค้าและเช็ก`
2. พิมพ์รหัสหรือชื่อคร่าว ๆ
3. เปิดดูรายการที่ระบบแนะนำ

## 6. สิ่งที่ STOCK ต้องระวัง

- ผลเทียบขึ้นกับจำนวนที่ใช้เทียบ อย่าลืมเช็ก quantity
- ถ้าไม่เจอสินค้า อาจลองพิมพ์ alias หรือชื่อร้านแทน
- ข้อมูลที่เห็นเป็นไปตาม branch scope อย่าตีความว่าของไม่มีถ้าอยู่นอกสิทธิ์ branch
